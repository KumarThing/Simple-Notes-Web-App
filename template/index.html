Note Web App

Action

Web Note App is a simple and practical web application that allows users to add notes through a form and view all saved notes on the same page. The app stores notes in a JSON file and automatically loads them when the server restarts, making the data persistent. This project is designed to be easy to understand while introducing real backend concepts like form handling, file storage, and server-side rendering.
